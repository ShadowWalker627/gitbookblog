代码题：
```(JavaScript)
function app() {
  setTimeout(() => {
    console.log("1-1");
    Promise.resolve().then(() => {
      console.log("2-1");
    });
  });
  console.log("1-2");
  Promise.resolve().then(() => {
    console.log("1-3");
    setTimeout(() => {
      console.log("3-1");
    });
  });
}
app();
```

结果
```
1-2
1-3
1-1
2-1
3-1
```

1. 先执行同步
2. 同一层级下(不理解层级，可以先不管，后面会讲),微任务永远比宏任务先执行
3. 每个宏任务,都单独关联了一个微任务队列
